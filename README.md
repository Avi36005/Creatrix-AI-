## 🧠 What is Creatrix AI?

Creatrix AI is a complete **AI-powered creator-economy platform** that solves two billion-dollar problems at once:

1. 🏢 **Brands** can't tell which influencers actually drive ROI — they rely on vanity metrics like follower count.
2. 🎬 **Creators** burn 80% of their time on research, scripting, editing, and captioning.

Creatrix fixes both with **ML-based influencer scoring on real data** and **autonomous viral-content generation**.

```
Type any creator  ──►  REAL data pulled live  ──►  ML scores + AI content  ──►  synced across every page
   (e.g. "mkbhd")        (YouTube Data API)        (XGBoost · Gemini · ElevenLabs)
```

## ✨ Key Features

### 🎯 Real Data, Not Mocks
- **Any creator, live** — type `mkbhd`, `MrBeast`, `Nike`, any channel → real subscribers, total views, video count, and engagement computed from actual recent uploads via the **YouTube Data API**.
- **`data_source` + `is_real` flags** on every response so the UI honestly shows **LIVE** vs **ESTIMATED**.
- **Keyless live trends** — Google News RSS + YouTube search, on-theme to the creator economy. No Reddit/News keys needed.

### 🔗 One Company, Synced Across Every Page
Analyze a creator once and the **same real data flows into Scorer, Growth, Authenticity, and Brand Matcher** automatically — backed by a shared `CreatrixCompany` store (persisted in `localStorage`) and a persistent status banner.

### 🧬 Real Machine-Learning Models
| Model | Algorithm | Purpose | Output |
|-------|-----------|---------|--------|
| **Ratefluencer Score™** | XGBoost ensemble | Campaign-success prediction | 0–100 |
| **Authenticity Detector** | IsolationForest | Fake-follower / bot / pod detection | Fake % + verdict |
| **Growth Engine** | Time-series projection | 3/6/12-month follower forecast | Projection curve |
| **Virality Predictor** | GradientBoosting | Expected views/likes/shares/saves | Score + grade |
| **Brand Matcher** | OpenAI embeddings + similarity (RAG) | Best-fit brand partnerships | Match score 0–100 |

### 🎬 End-to-End Content Automation
Trend → Script (hook/story/insight/CTA) → LinkedIn post → Instagram caption → **real ElevenLabs voiceover** → virality forecast — all from one dashboard.

### 🪄 In-App AI Assistant (Floating Orb)
A self-contained **floating assistant orb** lives on every page (`dashboard`, `index`, `tracks`). Ask it to explain a Ratefluencer Score, surface trends, or guide content generation — answers stream from **Groq Llama 3.3 70B** with an automatic **OpenAI GPT-4o-mini** fallback, and conversation context is kept across turns. Click 🔊 to hear any reply read aloud via **ElevenLabs** (voice plays only on demand, never auto-playing).

### 🛡️ Resilient Multi-Provider Fallback
Every LLM call degrades gracefully instead of failing: **chat** uses Groq → OpenAI; **rich generation** uses OpenAI GPT-4o-mini → Gemini 2.0 Flash → curated static text. One provider outage never takes the platform down.

## 📡 API Reference

**Base URL (production):** `https://creatrixai.web.app/api/v1` · **Docs:** [`/docs`](https://creatrix-ai-backend-3692981377.us-central1.run.app/docs)

### Track 01 — Influencer Intelligence
```http
POST /api/v1/influencer/analyze        # full report: metrics, scores, growth, authenticity, brands  (returns is_real + data_source)
POST /api/v1/influencer/growth         # follower/engagement growth forecast
POST /api/v1/influencer/authenticity   # fake-follower & bot detection
POST /api/v1/influencer/brands         # RAG brand matches
GET  /api/v1/reports                   # saved score reports (paginated)
POST /api/v1/reports/save              # persist a report
```

### Track 02 — Viral Creator Agent
```http
POST /api/v1/trends/discover           # LIVE trends (Google News + YouTube)  (returns is_live)
GET  /api/v1/trends/categories
GET  /api/v1/trends/{trend_id}
POST /api/v1/content/generate-script   # hook / story / insight / CTA
POST /api/v1/content/linkedin          # ready-to-post LinkedIn content
POST /api/v1/content/instagram         # caption + hashtags + CTA
POST /api/v1/virality/predict          # expected views/likes/shares/saves + score
POST /api/v1/virality/voiceover        # ElevenLabs MP3 (base64)
GET  /api/v1/virality/voices           # LIVE voices from the ElevenLabs account (no presets)
GET  /api/v1/reports/library           # saved content library
POST /api/v1/reports/library/save
```

### Grand Challenge — Autonomous Agent
```http
POST /api/v1/agent/run                 # SSE stream · 5-step LangGraph workflow
```

### In-App Assistant
```http
POST /api/v1/chat/                     # help chatbot · Groq → OpenAI fallback · keeps recent history
```

## 🔑 Environment Variables

| Variable | Required? | Powers | Source |
|----------|:---------:|--------|--------|
| `YOUTUBE_API_KEY` | ⭐ **recommended** | Real influencer data + live trends | [Google Cloud Console](https://console.cloud.google.com) (free) |
| `GEMINI_API_KEY` | ✓ | Analysis, captions, virality | [Google AI Studio](https://aistudio.google.com) |
| `GROQ_API_KEY` | ✓ | Reel scripts, trend ranking | [Groq Console](https://console.groq.com) (free) |
| `OPENAI_API_KEY` | ✓ | Brand-match embeddings (RAG) | [OpenAI](https://platform.openai.com) |
| `ELEVENLABS_API_KEY` | ✓ | Voiceover audio | [ElevenLabs](https://elevenlabs.io) |
| `RAPIDAPI_KEY` | optional | Real **Instagram** metrics | [RapidAPI](https://rapidapi.com) (paid) |
| `REDDIT_CLIENT_ID` / `NEWS_API_KEY` | optional | Extra trend sources | free — *not needed* (Google News is keyless) |

> 💡 **Graceful degradation:** every source falls back cleanly. With just a free `YOUTUBE_API_KEY`, influencer data **and** trends are fully real.
